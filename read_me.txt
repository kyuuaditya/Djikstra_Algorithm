how to use:
1. make sure you have a file named "maze.jpg".
2. run "main.exe".
you will see 2 windows, a terminal and a image as a result.

If want to solve a custom mazes:
1. go to https://www.pixilart.com/draw and select you height and width.
2. then click download and download png.
3. convert the file format from png to jpg on https://png2jpg.com .
4. save the image as maze.jpg where main.exe is present.
5. run "main.exe" to get the result.

Note: the maze must be a solvable one or else you might get a blank screen or garbage result.
